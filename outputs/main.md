# Awesome Open-Source MoE Models

A curated list of research papers on open-source **Mixture-of-Experts (MoE)** language models, covering foundational architectures, routing mechanisms, scaling strategies, and notable open-source releases.

**🛠️ Maintenance**

This repository is built and maintained using the [awesome paperlist maintainer](https://github.com/YichenZW/awesome-paperlist-maintainer). New papers are added regularly after manual auditing to ensure comprehensive coverage of the field.
